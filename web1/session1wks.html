    Exercises Now Grades Go Up

    1.

    The mathematical notation \(\displaystyle\lim \limits_{x \rightarrow 5} f(x)\) means \(\underline{ \qquad\qquad}\)
    1. we need to find the value of the function at \(x=5\text{.}\)
      🔗
      🔗
    2. we need to investigate the \(y\)-values of \(f(x)\) as \(x\) gets very near \(x=5\) from both the left and right sides of \(x=5\text{,}\) but not necessarily at \(x=5\text{.}\)
      🔗
      🔗
    3. we need to find the horizontal asymptotes.
      🔗
      🔗
    4. we need to find the instantaneous rate of change.
      🔗
      🔗
    5. we need to investigate the \(y\)-values as we approach \(x=5\) only on the right side.
      🔗
      🔗
    🔗
    🔗

    2.

    Find \(\displaystyle\lim_{x\rightarrow 2^-} \frac{3x^2}{x^2-4}\text{.}\) If the limit does not exist, but has infinite behavior, use limit notation to describe the infinite behavior of the function as \(x \longrightarrow 2^-\text{.}\)
    1. \(\displaystyle \displaystyle {\lim_{x\rightarrow 2^-} \frac{3x^2}{x^2-4}}=3\)
      🔗
      🔗
    2. \(\displaystyle \displaystyle{\lim_{x\rightarrow 2^-} \frac{3x^2}{x^2-4}}=-\infty\)
      🔗
      🔗
    3. \(\displaystyle{\lim_{x\rightarrow 2^-} \frac{3x^2}{x^2-4}}\) DNE, but has infinite behavior that can be described as \(\displaystyle{\lim_{x \rightarrow 2^-} f(x) \rightarrow -\infty}\text{.}\)
      🔗
      🔗
    4. \(\displaystyle{\lim_{x\rightarrow 2^-} \frac{3x^2}{x^2-4}}\) DNE, but has infinite behavior that can be described as \(\displaystyle{\lim_{x \rightarrow 2^-} f(x) \rightarrow \infty}\text{.}\)
      🔗
      🔗
    5. \(\displaystyle \displaystyle {\lim_{x\rightarrow 2^-} \frac{3x^2}{x^2-4}}=0\)
